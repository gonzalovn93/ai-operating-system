# Content

End-to-end AI content pipeline — from ideation to published carousel to performance retro.

## What's inside

| Workflow | What it does |
|----------|-------------|
| **content-package-generator** | Generates LinkedIn posts and 7-slide carousels from themes and raw ideas |
| **weekly-ideation** | Pulls from 80+ sources to surface content ideas aligned with voice and strategy |
| **content-retro** | Analyzes post performance metrics and surfaces patterns to improve future content |

## Stack
Claude API · Gemini · Python · LinkedIn metrics · Notion
