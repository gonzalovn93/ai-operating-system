# Career

AI-powered workflows to systematize job search, networking, and applications.

## What's inside

| Workflow | What it does |
|----------|-------------|
| **job-hunter** | Daily scan of target companies for new PM roles → writes matches to Notion |
| **networking-scout** | Finds relevant contacts at target companies via LinkedIn → suggests outreach |
| **application-blitz** | Analyzes job descriptions → generates tailored resumes and cover letters |
| **weekly-memo** | Synthesizes pipeline status, blockers, and next moves into a weekly strategy brief |

## Stack
Python · Claude Code · Notion API · Scheduled automations
